# 智言空间 - 网络舆情智能处理平台

## 项目介绍

**智言空间**是一个基于PySide6和前端技术的网络舆情智能处理平台，专注于网络信息真实性查询、不良言论甄别和虚假信息检测。

### 核心功能

- **网络信息真实性查询**：与权威数据库比对，评估信息可信度
- **不良言论甄别**：检测敏感词和不良言论
- **虚假信息检测**：识别网络谣言和虚假信息
- **网络热词识别**：识别网络流行语和隐晦表达

## 项目结构

```
├── main.py              # 完整版主界面
├── simple_ui.py         # 简洁版界面
├── dashboard.py         # 现代化仪表板
├── start.bat            # 启动脚本
├── requirements.txt     # 依赖配置
└── xiaochengxu/         # 微信小程序
    ├── index.html       # 首页
    ├── vocabulary.html  # 词汇检测
    ├── information.html # 信息真实性检测
    └── user.html        # 用户中心
```

## 运行环境

- Python 3.8+
- PySide6
- 其他依赖见 requirements.txt

## 快速开始

### 方法一：通过启动脚本运行

1. 双击 `start.bat` 文件
2. 选择对应的功能选项：
   - `1` - 运行界面启动器
   - `2` - 运行完整功能界面
   - `3` - 运行简洁界面
   - `4` - 运行现代化仪表板
   - `5` - 启动微信小程序前端页面
   - `6` - 安装依赖
   - `7` - 退出

### 方法二：直接运行

1. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

2. 运行完整版界面：
   ```bash
   python main.py
   ```

3. 运行简洁版界面：
   ```bash
   python simple_ui.py
   ```

4. 运行现代化仪表板：
   ```bash
   python dashboard.py
   ```

5. 查看微信小程序前端：
   直接打开 `xiaochengxu` 文件夹中的 HTML 文件

## 功能说明

### 完整版界面 (main.py)

1. **舆情综合分析**：
   - 文本输入和分析
   - 情感分析（正面/负面/中性）
   - 敏感词检测
   - 虚假信息检测
   - 风险等级判定
   - 实时预警
   - 结果导出

2. **敏感词专项检测**：
   - 独立敏感词扫描
   - 敏感词分类统计
   - 自定义词库
   - 批量检测

3. **虚假信息专项检测**：
   - 权威信息比对
   - 可信度打分
   - 虚假类型判定
   - 溯源提示
   - 二次校验机制

4. **舆情大盘监控**：
   - 舆情热度指数
   - 情感分布
   - 风险预警看板
   - 信源分布统计
   - 热点事件榜单

### 简洁版界面 (simple_ui.py)

- 快速分析（情感分析/敏感词检测/虚假信息检测）
- 高级设置（实时预警、自动保存、详细分析）
- 快捷操作（保存、清空、刷新、导出）
- 数据统计（分析次数、风险信息、敏感词数）

### 现代化仪表板 (dashboard.py)

- **现代化界面**：采用卡片式设计、侧边导航栏、顶部状态栏
- **多页面管理**：包含仪表板、用户管理、数据报表、系统设置、安全中心、文档中心、帮助支持
- **数据可视化**：卡片式数据展示、表格数据、报表生成
- **交互功能**：用户管理（增删改查）、数据导出、报表生成
- **响应式设计**：美观的界面布局和交互效果
- **舆情监控专用**：针对舆情监控场景优化的功能和数据展示
- **完善的设置界面**：系统设置和安全中心页面布局优化，确保内容完整可见

### 微信小程序前端 (xiaochengxu/)

1. **首页**：
   - 顶部导航（首页、词汇、信息、用户）
   - 搜索框（支持输入句子、新闻标题、热点词汇）
   - 热点事件展示（真实度标注、权威来源）
   - 信誉度显示

2. **词汇检测**：
   - 词汇输入查询
   - 安全性判定（是否为不良/敏感词汇）
   - 释义展示（网络流行词解释、安全等级）
   - 历史查询记录

3. **信息真实性检测**：
   - 输入新闻、消息、传言
   - 与权威数据库比对
   - 显示真实度百分比
   - 权威来源展示（包含相关图片）
   - 结果判定（真实信息、虚假信息、待定）

4. **用户中心**：
   - 登录（手机号、密码、验证码）
   - 个人信息
   - 设置（收藏、帮助中心）
   - 信誉度展示
